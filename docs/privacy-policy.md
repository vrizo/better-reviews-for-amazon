# Privacy Policy

This privacy policy applies to Better Reviews for Amazon.

## What the extension does

The extension adds a small review summary block on Amazon product pages.
It reads Amazon page content that is already visible in the browser and fetches Amazon review pages from the same Amazon site to calculate review totals.

## Data collection

The extension does not collect personal data for storage on a remote server.
It does not create user profiles.
It does not send analytics, tracking events, or advertising identifiers to any third party.

## Data sharing

The extension does not sell, share, or transfer user data to third parties.
It does not add affiliate links, ads, or tracking tags.

## Local storage

The extension stores cached review totals in the browser's local storage for up to 24 hours.
This cache stays on the user's device and is used only to reduce repeated requests to Amazon review pages.

## Scope

The extension is limited to Amazon pages listed in the extension manifest.
It is not designed for checkout, payment, addresses, or account settings.
