# AGENTS.md

## Project rules

- Write all documentation in simple English.
- Use `for Amazon`, not Amazon-branded naming.
- Do not imply endorsement, approval, or affiliation with Amazon.
- Do not collect user data.
- Do not add referral links, ads, or tracking.
- Keep the tools limited to Amazon pages.
- Put research and notes in `docs/`.
- Keep all shared translations in `src/i18n/locales/*.json`.
- Use the Amazon page language for text injected into Amazon pages.
- Use the browser language for manifest text and other browser-managed plugin metadata.
- Keep the same locale keys in every translation file.
- Use locale-aware typography in translations: real ellipsis, proper dashes, locale-appropriate quotes, and locale-specific spacing such as the required space before `?`, `!`, `:`, and `;` in French.
