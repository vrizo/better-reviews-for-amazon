# Naming Research

This file is only about naming.

## Preferred name

Use:

- `Better Reviews for Amazon`

This is safer than Amazon-branded naming because it describes what the tool is for instead of making Amazon part of the main brand.

## Naming rules

- Use `for Amazon` to show compatibility.
- Do not start the product name with `Amazon`.
- Do not use `official`, `approved`, or `authorized`.
- Do not use Amazon logos or visual branding.
- Add a disclaimer: `Not affiliated with or endorsed by Amazon.`

## Safe examples

- Better Reviews for Amazon
- Review Signals for Amazon
- Verified vs Vine Reviews for Amazon

## Risky examples

- Amazon Review Helper
- Amazon Vine Review Tool
- Official Amazon Vine Reviews

## Why this naming is safer

Amazon trademark guidance says Amazon marks should not be used in a way that implies endorsement.

Chrome Web Store policy also says extensions must not impersonate another company or mislead users about endorsement.

The `for Amazon` pattern is a simple compatibility label. It is easier to defend than using Amazon as the main brand.
